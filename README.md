Cavan Car-nage

A short text-based adventure game built with Java and JavaFX.

Overview

Cavan Car-nage puts the player in the middle of a small-town mystery: vital car parts have been stolen, and you need to get them back. You’ll fight, negotiate, and solve puzzles as you work through a network of shady NPCs, traders, and troublemakers. Every choice influences how characters react and what paths open up.

Type "help" for help

Features

Dynamic NPCs with different personalities, behaviours, and reactions

Inventory system with a range of items to acquire, trade, or use

Combat mechanics with varying outcomes based on stats and decisions

Puzzle interactions that require observation and smart choices

Narrative progression based on player actions

Tech Stack

Java 21

JavaFX

Maven project structure

Running the Game

Build the project using Maven:

mvn clean package


Run the generated JAR:

java -jar target/Cavan-Car-nage.jar


(If your JAR name differs, adjust the filename accordingly.)
